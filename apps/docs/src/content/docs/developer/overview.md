## Purpose of this section

This area is intentionally separate from the main user documentation.

Use it for:

- release-channel details
- deployment surfaces
- signing and update-chain notes
- internal project tracking pages

## Public user docs versus developer docs

The main guides explain how to use Pastiera as an app. This developer area exists for maintainers, contributors, and anyone who needs project-operational context.
