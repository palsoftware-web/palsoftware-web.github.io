## Scope

Pastiera includes Vietnamese Telex support implemented in early 2026.

## Setup

1. Install a build that includes PR `#144`.
2. Install/import the Vietnamese dictionary package required by that release line.
3. Open language/layout selector and choose Vietnamese Telex.

## Usage

- Type base Latin characters.
- Apply Telex modifiers according to Vietnamese Telex rules.
- Use suggestions/selection flow as usual for confirmation.

1. Verify Telex layout appears in language selector.
2. Test common vowel and tone combinations.
3. Confirm behavior in both plain text fields and apps with strict input handling.
