## Scope

Pastiera supports emoji search in two places:

1. Settings emoji picker (customization flow)
2. Inline emoji picker opened from SYM pages

## User behavior

1. Open an emoji picker.
2. Type a query using the hardware keyboard.
3. Results are matched across emoji names and keywords.
4. Clearing the query returns category browsing.

## Language behavior

- Search data is local/offline.
- Current bundled locale files include: `en`, `de`, `es`, `fr`, `hy`, `it`, `pl`, `ru`, `uk`.
- Fallback behavior uses English when no locale-specific terms are available.

## Inline picker key routing

- When inline picker search capture is active, hardware typing is routed into search.
- Typing is prevented from leaking into the target app text field.
- Tapping or moving cursor in app text fields disables capture and returns normal typing.
