## Scope

Clipboard history is integrated into the keyboard flow and exposed through status bar / SYM surfaces depending on configuration.

## Typical usage

1. Copy text in any app.
2. Open Pastiera clipboard view.
3. Select an entry to paste.
4. Pin entries you want to keep available.

## Notes

- Clipboard behavior is feature-rich and evolved over multiple merges.
- If expected entries are missing, first verify app/device clipboard restrictions and IME permissions.

## Troubleshooting checklist

1. Confirm clipboard feature is enabled in settings.
2. Confirm the relevant status bar button is enabled.
3. Re-test with text copied from another app in the same user session.
4. Check if device OEM policies limit clipboard background access.
