## Quali problemi risolve Pastiera

Pastiera e particolarmente utile se ti ritrovi in una o piu situazioni:

- la tastiera standard del tuo dispositivo hardware e troppo lenta o troppo limitata
- vuoi inserire simboli piu velocemente senza peggiorare la scrittura normale
- cambi spesso lingua o layout
- vuoi vere scorciatoie di navigazione su tastiera fisica
- vuoi clipboard, emoji e simboli nello stesso flusso di input
- vuoi mantenere una configurazione molto personalizzata anche dopo reinstallazioni o cambio dispositivo

## Setup: voglio digitare testo normale piu velocemente

Parti da:

- `Settings > Keyboard & Timing > Long Press Duration`
- `Settings > Smart Features > Auto-Capitalize First Letter`
- `Settings > Smart Features > Auto-Capitalize After Period`
- `Settings > Smart Features > Double Space to Period`
- `Settings > Smart Features > Clear Alt on Space`

## Setup: voglio che il long press usi le variazioni invece di Alt

Imposta `Settings > Keyboard & Timing > Long Press Modifier` su `Variations`.

E utile se:

- scrivi spesso lettere accentate
- preferisci varianti di carattere invece di simboli guidati da Alt
- vuoi umlaut o caratteri simili piu vicini alla lettera base

## Setup: voglio umlaut tedeschi e sharp s su QWERTZ

Usa il layout `German (QWERTZ, multi-tap)` in `Settings > Languages and Maps`.

L'asset attuale include multi-tap per:

- `a -> ä`
- `o -> ö`
- `u -> ü`
- `s -> ß`

Nota pratica: il multi-tap e un input basato sul timing. Se premi la stessa lettera molto velocemente, serve spesso un breve periodo di adattamento.

## Setup: voglio Vietnamese Telex

Pastiera include ora una modalita dedicata Vietnamese Telex.

1. Apri `Settings > Languages and Maps`.
2. Aggiungi o modifica l'input style vietnamita.
3. Scegli il layout Telex.
4. Prova vocali e toni in un normale campo di testo.

## Setup: voglio pagine SYM ordinate e prevedibili

Apri `Settings > Customization > Customize SYM Keyboard`.

Li puoi decidere:

- quali pagine SYM restano attive
- se il clipboard entra nel ciclo
- se il picker emoji completo entra nel ciclo
- l'ordine delle pagine
- se SYM si chiude automaticamente dopo l'inserimento

## Setup: voglio un profilo piu orientato a utility o coding

Una combinazione pratica spesso e:

- `Long Press Modifier = Sym` o `Variations`
- barra variazioni statica
- ordine SYM pulito
- Nav Mode configurato sui tasti di movimento preferiti
- scorciatoie launcher o power solo dopo che il resto e stabile

## Setup: quando scrivo veloce si attiva spesso la cosa sbagliata

Di solito serve una piccola combinazione di aggiustamenti:

1. regolare `Long Press Duration`
2. decidere se il long press debba davvero essere `Alt`, `Sym` o `Variations`
3. controllare se il layout attivo usa multi-tap
4. semplificare le pagine SYM per ridurre i layer in competizione
