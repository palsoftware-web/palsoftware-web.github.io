## Sezione Auto-correction

In `Settings > Auto-correction` trovi tutto cio che riguarda suggerimenti e sostituzioni.

Funzioni attuali:

- set di sostituzioni per lingua
- regole globali tramite ricette Pastiera
- attivazione/disattivazione delle lingue
- modifica delle singole correzioni
- gestione del dizionario utente
- regolazione del motore suggerimenti

## Feature set attuale

La versione corrente include:

- `Suggestions while typing`
- `Match accented words`
- `Auto-replace on space/enter`
- `Maximum correction distance`
- `Manage user dictionary`
- `Keyboard Proximity Ranking`
- `Edit Type Ranking`
- `Experimental suggestions`

## Dizionario utente

Nel dizionario utente puoi:

- aggiungere parole
- modificare parole salvate
- eliminare voci
- mantenere termini personali non presenti nei dizionari standard

## Lingue e fonti di correzione

La qualita dei suggerimenti dipende dalle risorse disponibili per il locale attivo. Un layout puo essere utilizzabile anche se il supporto suggerimenti per quella lingua e limitato.

## Stato sperimentale

Il motore suggerimenti piu recente resta considerato sperimentale. In pratica:

- il comportamento puo cambiare piu rapidamente del core di digitazione
- la qualita dipende molto da lingua e layout
- una configurazione troppo aggressiva non e adatta a tutti i casi

## Consiglio pratico

1. Attiva i suggerimenti durante la digitazione.
2. Mantieni moderato l'auto-replace.
3. Aggiungi presto i tuoi termini al dizionario utente.
4. Aumenta l'aggressivita delle correzioni solo quando la lingua principale funziona bene.
